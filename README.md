# Fish Disease Detection using Deep Learning

This project explores the use of deep learning models to detect fish diseases from images.

The notebooks implement convolutional neural networks using transfer learning models such as MobileNet and VGG16 for image classification.

## Models Used

MobileNet  
VGG16  

These models were used to classify fish disease images and evaluate model performance.

## Technologies Used

Python  
TensorFlow / Keras  
Jupyter Notebook  
Deep Learning
